# apexplanet-data-analytics
# Task 1: Foundational Setup & Exploratory Data Analysis (EDA)

## Objective

Set up the analytics environment, clean the dataset, and perform EDA to identify insights.

## Dataset

Global E-commerce Sales Dataset

## Tools & Technologies

Python
Jupyter Notebook
Pandas
NumPy
Matplotlib
Seaborn
GitHub

## Work Completed

### Environment Setup

* Installed Python and required libraries.
* Created project structure and GitHub repository.

### Data Understanding

* Loaded dataset into Pandas.
* Analyzed columns, data types, and dataset information.
* Documented data source and limitations.

### Data Cleaning

* Handled missing values.
* Removed duplicates.
* Fixed data types.
* Standardized column names.
* Detected and handled outliers.

### Exploratory Data Analysis (EDA)

* Generated statistical summaries.
* Created histograms, boxplots, bar charts, scatter plots, and heatmaps.
* Identified trends, patterns, and anomalies.

## Key Findings

* Most sales fall within a moderate range.
* Some product categories perform better than others.
* Sales and profit show a positive relationship.
